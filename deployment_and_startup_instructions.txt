Инструкция по развертыванию и запуску

Развертывание:
1. Установить Docker на вашу систему. Для этого можно использовать официальный сайт Docker(https://www.docker.com/)
2. Создать новую директорию для проекта и открыть ее в командной строке или терминале.
3. В docker-compose.yml заменить your_database_name, your_username, your_password и your_root_password на соответствующие значения для вашей базы данных
4. Перейти в командной строке или терминале в директорию с проектом и выполнить команду docker-compose up. Docker скачает необходимые образы и запустит контейнеры
5. После успешного запуска контейнеров необходимо обратиться к скрипту по адресу http://localhost:8080/?user_id=10, чтобы получить баланс пользователя в формате JSON
6. В случае, если порт 8080 будет занят(такое бывает), его можно изменить в файле "docker-compose.yml"

Получение баланса пользователя:
1. Открыть веб-браузер и ввести URL-адрес http:localhost:8080/?user_id=10
2. Нажать "Enter" или выполнить запрос, чтобы отправить его на сервер
3. Сервер обработает запрос и вернет ответ в формате JSON
4. Веб-браузер отобразит ответ на экране. Если пользователь с указанным user_id будет найден, то мы увидим баланс пользователя. Например: {"balance": "10"}. Если пользователь не будет найден, то ошибка с соответствующим сообщением. Например: {"error": "User not found"}
