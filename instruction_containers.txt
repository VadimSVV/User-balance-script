Инструкция по запуску контейнеров:
1. Чтобы запустить контейнеры нужно выполнить команду "docker-compose up" в директории проекта
2. В случае успешного запуска контейнеров следует обратиться к скрипту "user_id.php" по адресу http://localhost:8080/?user_id=10, чтобы получить баланс пользователя в формате JSON
3. Для работы скрипта требуется наличие расширения PDO. Это необходимо для работы с базой данных MySQL
